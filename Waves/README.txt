waves.conf mit einem Editor öffnen und eventuell den 'directory' Pfad anpassen
Dort wird die Waves Blockchain gespeichert (in diesem Fall unter C:/Waves)

wenn Java installiert ist, kann die Blockchain mit dem Befehl 
	java -jar waves-all-0.10.2.jar waves.conf 
gestartet werden

Der Download dauert einige Stunden
